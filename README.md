# Projet CROUS – Analyse des services de restauration

Projet d'analyse des services de restauration du CROUS (Lille & Valenciennes).

## Données

- Source : data.enseignementsup-recherche.gouv.fr
- Format : fichier Excel (.xlsx)

## Objectifs

- Identifier les types de services CROUS
- Analyser la répartition géographique
- Étudier les horaires et localisations

## Outils

- Python
- Pandas
- Matplotlib
- Jupyter Notebook
